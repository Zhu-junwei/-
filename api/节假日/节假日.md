

# 节假日
http://blog.bitefu.net/post/4.html


| 参数名  | 必选  | 类型     |  说明   |
|------|-----|--------|-----|
| d    | 是   | string |  日期   |
| back | 否   | string |  	返回格式(json)   |
| info | 否   | string |   返回详细信息(适用单日期/单月)  |



```
http://tool.bitefu.net/jiari/?d=2022-10-26
```

```
http://tool.bitefu.net/jiari/?d=2022-10-26&back=json&info=1
```
