几个简单的快捷键

`Tab` 

- 单击：命令提示
- 双击：所有可能的选项

```
#单击后补齐date
[zjw@centos7 ~]$ dat

#双击
[zjw@centos7 ~]$ ls -al .bash
.bash_history  .bash_logout   .bash_profile  .bashrc  
[zjw@centos7 ~]$ date --
--date        --help        --reference=  --rfc-3339=   --universal
--date=       --iso-8601    --rfc-2822    --set=        --version
```

`Ctrl`+`d` : 离开命令行，相当于exit

`Shift`+[`PgUp`|`PgDn`] : 向上、向下查看输出信息
